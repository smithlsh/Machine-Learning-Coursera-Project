I. Introduction

In this paper we decribe the use of the use of the random forest model to predict barbell lifts. The random forest model is an example of a tree based classicification method. In a tree based classicfication method, the space spanned by the individual predictors is divided into discrete regions represented by nodes (or leaves) which in turn are subdivided into other regions. Eventually, this process comes to halt and a response is predicted based on the the most commonly occurring class of training obseravations that occurs in the node. Our results show that the rndom forest model can be used to make accurate predictions on the whethe the barbell lifts are done correctly.

III. Error Estimation

An important part of validationg one’s model is to estimate the test error without actually running it on the test data. The essential idea is hold out a subset of the training is the model fitting process and use this data to estimate the test error rate. In the random forest approach instead of explicitly using cross-validation to estimate the test error, one takes advantage of bootstrapping (James et al p. 317) With bootstrappng, one gets “b” predictions which are then averaged. Further, since each bootstrapped sample uses about 2/3’s of the observations, the remaining one third of the oberservations (the so called “out of bag” observations) are used to estimate the test error. Since these responses were produced by trees which were not used to fit the data, this is a statistically valid way to estimate the error.

The small class errors in the confusion matrix, the low overall out of bag error rate and plots shown below all indicate that the random forest algorithm should give extremely valid predictions on the test data.

IV. Variable Importance

One of the advantages of simple tree based methods is that there is one resulting decision tree which, perhaps after some pruning, makes it easy to determine which variables are important. In random forest multiple tree are used making it difficult to determine which variables are important. Fortunately, random forest provides a simple way to assess the importance of each predictor based on the mean decrease in accuracy and the mean decrease in the GINI coefficient due to splits over a given predictor. The larger the value the more important the predictor. The table below shows the mean decrease in the accuracy and GINI coefficients:
